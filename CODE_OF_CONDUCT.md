# Código de Conducta

## Nuestro compromiso

Este repositorio es un espacio técnico abierto. Cualquier persona que contribuya, reporte un issue o participe en debates se compromete a mantener un entorno respetuoso y constructivo.

---

## Comportamiento esperado

- Comunicación técnica clara y respetuosa
- Aceptar críticas constructivas sobre el código
- Dar crédito a contribuciones externas
- Usar los Issues para problemas técnicos, no para debate fuera de contexto

---

## Comportamiento inaceptable

- Lenguaje ofensivo o descalificador
- Spam o promoción no relacionada con la herramienta
- Apropiación de código sin mantener la licencia GPL-2.0

---

## Aplicación

Las contribuciones que no respeten este código serán rechazadas. Los Issues que no sean técnicos serán cerrados sin respuesta.
